# نظام إدارة الأنشطة الطلابية - جامعة العين

## مرحبا بك في نظام جامعة العين لإدارة الأنشطة الطلابية

هذا المشروع هو نظام شامل لإدارة الأنشطة الطلابية في جامعة العين. يوفر للطلاب والموظفين إمكانية:
- تسجيل الدخول والتسجيل
- عرض الأنشطة المتاحة
- التسجيل في الأنشطة
- إرسال الرسائل والملفات
- إدارة طلبات الموظفين

## المتطلبات

- Python 3.8+
- pip (مدير الحزم)

## التثبيت والإعداد

### 1. إنشاء بيئة افتراضية

```bash
python -m venv venv
```

### 2. تفعيل البيئة الافتراضية

**على Windows:**
```bash
venv\Scripts\activate
```

**على Mac/Linux:**
```bash
source venv/bin/activate
```

### 3. تثبيت المتطلبات

```bash
pip install -r backend/requirements.txt
```

### 4. إعداد قاعدة البيانات

```bash
cd backend
python manage.py migrate
```

### 5. إنشاء حساب المسؤول (اختياري)

```bash
python manage.py createsuperuser
```

### 6. إنشاء بيانات تجريبية (اختياري)

```bash
python manage.py seed_super_employee
```

## تشغيل المشروع

### تشغيل خادم Django

```bash
cd backend
python manage.py runserver
```

سيتم تشغيل الخادم على: **http://127.0.0.1:8000/**

### الوصول إلى الموقع

1. افتح المتصفح
2. انتقل إلى: http://127.0.0.1:8000/
3. قم بتسجيل الدخول أو إنشاء حساب جديد
